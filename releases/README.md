# Mindra — Descargas

Descarga la última versión de la app Mindra para Android.

## Última versión

| Archivo | Descripción |
|---------|-------------|
| `mindra-latest.apk` | Versión más reciente (Android 5.0+) |

## Instalación

1. Descarga el archivo `.apk`
2. En tu Android ve a **Ajustes → Seguridad → Fuentes desconocidas** y actívalo
3. Abre el archivo descargado e instala
